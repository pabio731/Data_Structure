

트리는 계층 구조 자료를 저장하는 자료구조이다.
계층 구조란 트리를 구성하는 노드들이 부모-자식의 관계가 있다는 것이다.
즉 특정 무모 노드 하나에 여러가지 자신 노드들이 붙는다는 것.

또한 하나의 노드에 여러 개의 노드가 붙을 수있기 때문에 비선형 자료구조이다


트리의 용어 정리

루트(root) 노드 : 트리에서 첫 번째. 가장 부모가 되는 노
단말(leaf) 노드 : 트리에서 가장 마지막 노드. 자식 노드가 없음
형제(siblin) 노드 : 같은 부모를 가지는 노드

노드의 레벨 : 루트 노드로부터의 거리
노드의 높이 : 루트 노드로부터 가장 먼 거리에 있는 자식 노드 높이 +1
노드의 차수(degree) : 자식 노드의 갯수

이진 트리 : 자식의 갯수를 최대 2개 까지만 가지는 트리
 - 포화 이진 트리 : 모든 레벨의 노드가 꽉 차있는 트리
 - 완전 이진 트리 : 높이가 h인 트리에서 레벨 1부터 h-1까지는 포화이진 트리이고
				   h에서는 왼쪽부터 채워져 있지만, 꽉 차있지는 않을 수 있는 트리


이진 트리의 추상자료형

이름											Input				Output
이진 트리 생성			MakeBinTree()		원소 e				이진트리bt
루트 노드 반환			GetRootNode()		이진트리bt			루트노드rn
왼쪽 자식 노드 추가		InsertLeftChild()	부모노드pt, 원소e		생성된 노드n
오른쪽 자식 노드 추가		InsertRightChild()	부모노드pt, 원소e		생성된 노드n
왼쪽 자식 노드 반환		GetLeftChild()		노드 n				왼쪽자식 ln
오른쪽 자식 노드 반환		GetRightChild()		노드 n				오른쪽 자식rn
루트 노드의 데이터		GetData()			이진트리bt			원소e
이진 트리 삭제			DeleteBinTree()		이진트리bt
